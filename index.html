Programming Language Experience

C#

bahasa pemrograman yang dikembangkan oleh Microsoft. C# merupakan bahasa pemrograman berorientasi objek (OOP) yang serbaguna. C# dapat digunakan untuk mengembangkan berbagai program, seperti aplikasi seluler, permainan video, dan perangkat lunak perusahaan.

Python

bahasa pemrograman tingkat tinggi yang dikembangkan oleh Guido van Rossum pada tahun 1991, dikenal dengan sintaksnya yang mudah dibaca dan fleksibel, serta digunakan secara luas dalam berbagai bidang seperti pengembangan web, analisis data, dan kecerdasan buatan.

CSS

CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mengatur tampilan dan format halaman web. CSS dapat digunakan untuk mengatur warna latar, ukuran font, spasi antar paragraf, hingga tata letak halaman.

Java Script

bahasa pemrograman yang digunakan untuk membuat halaman web yang interaktif. JavaScript merupakan salah satu bahasa pemrograman paling populer di dunia, terutama untuk pengembangan web.
